## Spacing Scale

### Admin Dashboard (Tailwind / shadcn)

Tailwind default scale (multiples of 4):
| Token | Value | Usage |
|-------|-------|-------|
| p-1 / gap-1 | 4px | Icon gaps, compact badges |
| p-2 / gap-2 | 8px | Input group spacing, pill padding |
| p-4 / gap-4 | 16px | Card padding, form field gaps (DEFAULT) |
| p-6 / gap-6 | 24px | Section spacing, modal padding |
| p-8 / gap-8 | 32px | Page section gaps |
| p-12 | 48px | Hero section padding |
| p-16 | 64px | Page-level vertical rhythm |

Sidebar: 256px (w-64), collapsed: 64px (w-16). Header: 64px (h-16).

### Mobile (React Native)

Base grid: 16px. All values multiples of 4.
| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon-to-text gap, dot indicators |
| sm | 8px | Card list gaps, chip padding |
| md | 16px | Card internal padding, element spacing (DEFAULT) |
| lg | 20px | Screen horizontal padding |
| xl | 24px | Section gaps |
| 2xl | 32px | Bottom CTA spacing, safe area offset |
| 3xl | 48px | Hero section padding |

Exceptions: bottom tab nav height — 72px (includes safe area).

## Color

### Admin Dashboard

Neutral palette (shadcn default):

| Role | Value | Usage |
|------|-------|-------|
| Background | `hsl(0 0% 100%)` (white) | Page background |
| Foreground | `hsl(240 10% 3.9%)` (near-black) | Text |
| Card | `hsl(0 0% 100%)` | Card backgrounds |
| Muted | `hsl(240 4.8% 95.9%)` | Secondary bg, disabled states |
| Primary | `hsl(240 5.9% 10%)` | Primary buttons, active states |
| Destructive | `hsl(0 84.2% 60.2%)` | Delete buttons, error states |
| Border | `hsl(240 5.9% 90%)` | Card borders, input borders |
| Success | `hsl(142 71% 29%)` | Connection test success, green badges |
| Warning | `hsl(38 92% 50%)` | Sync warnings, attention badges |

### Mobile (BizPulse brand — from UI Guide)

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#E8F4FD` → `#C2E0F8` (gradient) | Screen backgrounds |
| Secondary (30%) | `#FFFFFF` at 90% opacity | Cards (glass-morphism) |
| Accent (10%) | `#6C63FF` → `#3B9FE8` (gradient) | AI elements, CTA buttons, active nav |
| Deep accent | `#0A2540` | Hero cards, splash screen, dark UI |
| Text primary | `#0A2540` (navy) | All body and heading text |
| Text secondary | navy at 50% | Labels, secondary info, timestamps |
| Success | `#00C896` | Up arrows, positive deltas, connected status |
| Warning | `#FFB020` | Dead stock alerts, reorder signals |
| Danger | `#FF4757` | Revenue drops, churn warnings, error |
| AI signature | `#6C63FF` → `#3B9FE8` | ✦ sparkle prefix, AI content left border |

Accent reserved for: CTA buttons, AI content markers (✦ border/badge), active navigation tab, progress indicators. Never for: body text, card backgrounds, status indicators (use semantic colors).

## Interactive States

Every interactive element must handle these states:

| State | Visual | Applies to |
|-------|--------|------------|
| Default | Base styles | All elements |
| Hover | Background shift, cursor pointer | Desktop: buttons, links, table rows |
| Focus | Ring (2px, primary color, offset) | Inputs, buttons, links (keyboard nav) |
| Active/Pressed | Scale-down (mobile), darker bg (desktop) | Buttons, list items |
| Disabled | 50% opacity, no pointer events | Buttons, inputs, form submits |
| Loading | Spinner replaces text, button disabled | Submit buttons, test connection |
| Error | Red border, error text below | Inputs, connection test result |
